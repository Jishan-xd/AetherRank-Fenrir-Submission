AetherRank: AI-Powered Esports Platform

Build a locally runnable AI-powered esports platform called "AetherRank". The platform should help players participate in tournaments, build teams, track performance, and improve their skills through intelligent analytics. AetherRank's unique feature is AetherMind™, a personal AI gaming companion that learns from previous matches and provides personalized guidance during gameplay.

Required experience:

- User registration, authentication, and profile management.
- Player dashboard displaying rankings, achievements, match history, and performance statistics.
- Tournament creation, registration, scheduling, and leaderboard functionality.
- Team creation, recruitment, and team management system.
- Global and regional player rankings.
- Match analytics including win rate, performance trends, and skill progression.
- Achievement and reward system.
- Responsive user interface for desktop and mobile devices.
- Dedicated AetherMind™ assistant panel integrated within the platform.

AetherMind™ requirements:

- Analyze player match history and performance data.
- Identify recurring gameplay mistakes and weaknesses.
- Generate personalized coaching recommendations.
- Provide contextual suggestions during live gameplay.
- Track player improvement over time.
- Continuously improve recommendation accuracy through adaptive learning.
- Display AI confidence level and learning progress.

Final deliverable:

- Tournament management system.
- Team recruitment and team management functionality.
- Performance analytics dashboard.
- Global and regional leaderboards.
- Achievement and progression tracking system.
- AetherMind™ adaptive AI coaching system.
- Responsive esports-inspired user interface.
- Complete locally runnable web application with setup instructions.
